# 🎬 ReelGenerator

Generador de Reels — corta vídeos largos en clips verticales (9:16) listos para publicar,
con aceleración por GPU (NVIDIA / Intel / AMD).

## 📥 Descargar

Ve a [**Releases**](../../releases/latest) y descarga:

- **`ReelGenerator-Setup-X.X.X.exe`** — instalador (recomendado, recibe actualizaciones automáticas)
- **`ReelGenerator-Portable-X.X.X.exe`** — versión portable, sin instalación

## 🔑 Activación

1. Abre ReelGenerator. Verás la pantalla de activación con tu **ID de máquina**.
2. Pulsa **Copiar** y envíaselo a tu proveedor.
3. Recibirás un código de licencia: pégalo y pulsa **Activar**.

## ⚠️ Aviso de Windows SmartScreen

Al abrir el instalador, Windows puede mostrar *"Windows protegió tu PC"*.
Haz clic en **Más información → Ejecutar de todas formas**.
Esto ocurre porque el ejecutable aún no está firmado digitalmente.

## 💻 Requisitos

- Windows 10/11 (64 bits)
- GPU NVIDIA/Intel/AMD recomendada (también funciona con CPU)

---
*Las actualizaciones se instalan automáticamente: la app te avisa cuando hay una versión nueva.*
